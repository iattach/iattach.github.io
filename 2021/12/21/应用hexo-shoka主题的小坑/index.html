# lean cloud иҜ„и®ә

дёҖејҖе§Ӣи®ҫзҪ®зҡ„йғҪеҜ№зҡ„пјҢдҪҶе°ұжҳҜеҸ‘дёҚдәҶиҜ„и®әгҖӮжҹҘдәҶдёӢеӣ дёәжҳҜеӣҪйҷ…зүҲ lean cloudпјҢжІЎи®ҫзҪ®дё»йўҳдёӯ valine зҡ„ serverURLsгҖӮиҝҷйҮҢеә”и®ҫзҪ®дёәе®ҳзҪ‘з»ҷзҡ„ REST API жңҚеҠЎеҷЁең°еқҖгҖӮ

# hexo

# еҒ·жҮ’ bat и„ҡжң¬

жғіеҒ·зӮ№жҮ’зӣҙжҺҘдёҖй”®еҸ‘еёғпјҢе°ұеҶҷдәҶдёӘ bash и„ҡжң¬гҖӮз„¶еҗҺеҸҲиё©еқ‘дәҶгҖӮгҖӮгҖӮ

иө·и„ҡжң¬еҗҚеӯ—дёә hexo.bat, з»“жһң.bat и„ҡжң¬дёҚж–ӯйҮҚеӨҚжү§иЎҢеҗҢдёҖе‘Ҫд»ӨгҖӮеӣ дёәдёҺзі»з»ҹ hexo иҝҷдёӘе‘Ҫд»ӨеҶІзӘҒпјҢжүҖд»ҘйңҖиҰҒиө·дёӘдёҚеҶІзӘҒзҡ„еҗҚеӯ—гҖӮ

жғіиҰҒжҳҫзӨәе‘Ҫд»Өзҡ„жү§иЎҢз»“жһңпјҢжңүзҡ„дәәиҜҙеҠ дәҶ pause е°ұиЎҢпјҢжңүдәәиҜҙеҠ дәҶ cmd /f k е°ұиЎҢгҖӮе…¶е®һйғҪжІЎз”ЁпјҢйңҖиҰҒеҶҷ cmd /c "е‘Ҫд»Ө"гҖӮиҝҷж ·жңҖеҗҺиҝҳиғҪжҳҫзӨәд»»ж„Ҹй”®йҖҖеҮәпјҢеӨ§дёәйңҮжғҠгҖӮ

жіЁж„Ҹпјҡalgolia еә”еңЁ generate д№ӢеҗҺпјҢеӣ дёә algolia йңҖиҰҒ generate дә§з”ҹзҡ„ж–Үд»¶
